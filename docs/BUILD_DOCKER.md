# Инструкция по сборке докер-образа

### Характеристики целевого окружения
При подготовке образа полезно учитывать архитектуру и мощности суперкомпьютера:

* **Операционная система:** Ubuntu 24.04 LTS.
* **Процессор:** 256 ядер.
* **Оперативная память:** 755.5 GiB.
* **Видеоускорители:** 4 устройства.
* **Драйвер и CUDA:** NVIDIA Driver Version 575.64.03, CUDA Version 12.9.
* **Движок контейнеризации:** Docker Engine (Community) версии 28.4.0.


### Шаг 1. Локальная сборка.
В директории, где лежат Dockerfile и requirements.txt, выполните команду для сборки образа (обязательно укажите архитектуру linux/amd64):
```
docker build --platform linux/amd64 -t supercomp-build .
```
### Шаг 2. Сохранение в архив.
Для переноса на суперкомпьютер запакуйте образ в tar-архив:
```
docker save -o supercomp-build_latest.tar supercomp-build:latest
```
### Шаг 3. Перенос и загрузка на сервере.
После того как вы перекинете .tar файл и папку yolo_dataset на суперкомпьютер, восстановите образ командой (флаг -i обязателен):
```
docker load -i supercomp-build_latest.tar
```
### Шаг 4. Финальный запуск.
Чтобы запустить контейнер, пробросить порт Юпитера, задействовать все видеокарты и примонтировать код с датасетом, используйте эту команду (обратите внимание на параметр --shm-size=8g ):
```
docker run --gpus all -it --shm-size=8g -p 8888:8888 -v $(pwd):/usr/app/ supercomp-build:latest
```
(Здесь флаг -v $(pwd):/usr/app/ пробрасывает вашу текущую папку на сервере внутрь контейнера. Запускайте эту команду, находясь в папке, где лежит ваш блокнот и датасет).

В терминале появится ссылка вида http://127.0.0.1:8888/?token=.... Скопируйте ее, вставьте в браузер (поменяв 127.0.0.1 на IP-адрес суперкомпьютера), откройте свой ноутбук с кодом и запускайте обучение на всех 256 ядрах и 4 GPU. По завершении работы контейнера все ресурсы суперкомпьютера будут корректно освобождены.
